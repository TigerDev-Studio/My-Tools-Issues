# TigerDev Studio Tool Support

Public issue tracker for TigerDev Studio developer tools and VS Code extensions.

The source code for some tools is private. This repo is only for collecting bugs,
feature requests, questions, screenshots, logs, and reproduction details in one
public place.

## Report An Issue

Use the issue form that matches your request:

- [Bug report](https://github.com/TigerDev-Studio/My-Tools-Issues/issues/new?template=bug.yml)
- [Feature request](https://github.com/TigerDev-Studio/My-Tools-Issues/issues/new?template=feature_request.yml)
- [Question](https://github.com/TigerDev-Studio/My-Tools-Issues/issues/new?template=question.yml)
- [Choose an issue type](https://github.com/TigerDev-Studio/My-Tools-Issues/issues/new/choose)

## Supported Tools

- GitHub SSH Profile Switcher
- Frontend Helper Pro
- AI Git Commit Assistant
- Smart Regex Builder
- Git Graph Pro
- Folder Structure Generator

## Before Posting

Do not share private data in public issues.

Remove these from logs, screenshots, and examples before posting:

- Access tokens
- Passwords
- SSH private keys
- Private repository URLs
- Customer or company secrets
- Personal data that is not required to debug the issue

For bugs, please include:

- Tool name
- Tool version
- VS Code version
- Operating system
- Steps to reproduce
- Expected result
- Actual result
- Logs or screenshots, with secrets removed

## Private Repositories

Private source repositories stay private. If an issue requires source-code
changes, it will be tracked here publicly and fixed in the related private repo.
